Earsdrop

Earsdrop is an Android app that allows users to tune in to strangers’ music in real time. It requires that both users have the application installed, and have Spotify Premium accounts. One user is the broadcaster, sending out their music live to users around them, and the other is the listener, tuning in to the broadcaster’s music. It works in the background for the broadcaster, all they need to do is play music through Spotify as normal. The listener simply chooses which user around them they would like to listen to (within a half mile radius), and get to listen to the broadcaster’s current track at the same position as the broadcaster.

View Project

Custom Mixing MIDI Controller

A MIDI Controller made with a Teensy 3.2 board + A VST plugin enabling its use in Ableton Live. The VST was created with JUCE, using Maximilian for synthesis. The goal of the controller and its plugin is to allow scratching of audio clips through Ableton for live performance use. Instead of having to choose between mixing through a program like Serato or Tracktor or performing using a program like Ableton, you can now use the controller to trigger and manipulate audio clips while performing live.
